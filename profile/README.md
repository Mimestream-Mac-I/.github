## Overview

<a href="#">Mimestream</a> is a native Mac email client built exclusively for Gmail that provides the full Gmail feature set — labels, filters, conversation threading, and keyboard shortcuts — through a purpose-built macOS interface that feels nothing like a browser tab.

The <a href="#">Gmail API integration</a> accesses your inbox through Google's official API rather than IMAP, which means labels work as labels rather than folders, filters apply immediately, and Gmail-specific features like muting threads and marking as spam behave exactly as they do in Gmail's own interface. <a href="#">Full label hierarchy</a> displays nested labels in the sidebar with unread counts, and any label applied in the web interface appears instantly in Mimestream.

<a href="#">Conversation threading</a> groups messages by subject in the same way Gmail does, keeping related exchanges together while allowing you to read and reply without losing context. Expand and collapse individual messages within a thread. <a href="#">Inline reply</a> composes responses directly beneath the message you're replying to without opening a separate composition window.

The <a href="#">keyboard-first navigation</a> covers every common action — archiving, labeling, starring, replying, forwarding, and moving between messages — through single-key shortcuts that match Gmail's own keyboard scheme. Users who navigate Gmail entirely by keyboard will find the same shortcuts work immediately without remapping. <a href="#">Offline access</a> makes recent messages readable and composable without an internet connection, syncing changes when connectivity returns.

## Additional Information

Gmail users who have tried other Mac email clients know the frustration: the client either treats Gmail labels as folders and breaks the mental model entirely, or it uses IMAP in a way that makes filters and labels feel like afterthoughts. Mimestream was engineered from the start to use the Gmail API, which means it doesn't have to approximate Gmail behavior — it is Gmail behavior, wrapped in a Mac application that respects how macOS applications are supposed to work. The result is the first email client that Gmail power users can switch to without feeling like they've downgraded.
